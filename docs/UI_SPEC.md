# AI Companion 管理后台 - 产品规格

> 版本: v1.0
> 日期: 2026-04-26

---

## 一、产品概述

### 1.1 定位

AI Companion 管理后台 — 面向开发者和运维人员的控制面板，用于监控 Bot 运行状态、管理会话和上下文、查看日志、配置参数。

### 1.2 核心价值

- **可观测性**：实时了解 Bot 运行状态
- **可控性**：配置管理、进程启停
- **可调试**：日志追踪、上下文检查
- **可维护**：记忆管理、数据清理

## 四、页面详情

### 4.1 监控仪表盘

#### 功能描述

展示 AI Companion 系统的实时监控数据，帮助用户快速了解 Bot 运行状态。

#### 交互流程

```
打开页面 → 自动加载数据 → 每5秒刷新 → 用户可手动刷新
```

#### 数据来源

- 系统指标：Rust 后端读取 `/proc` 或 `sysinfo`
- Bot 指标：Rust 调用 Python Core 读取 SessionStore
- 上下文：Rust 读取 MemoryEngine 统计

#### 展示内容

| 区块 | 内容 | 更新频率 |
|------|------|----------|
| 状态卡片 | 运行状态、运行时长、今日对话数、主动消息数 | 5秒 |
| 上下文状态 | 使用量/上限百分比、压缩次数 | 5秒 |
| Token消耗 | 今日消耗/限额、延迟 | 5秒 |
| 主动唤醒 | 启用状态、阈值、今日已发、情绪触发次数 | 5秒 |
| 记忆状态 | 三层记忆条数 | 10秒 |

#### 刷新机制

- 自动刷新：每 5 秒轮询
- 手动刷新：点击刷新按钮
- 页面隐藏：停止刷新，页面可见时恢复

### 4.2 会话管理

#### 功能描述

管理 Bot 的会话生命周期，查看会话详情，执行重置/挂起操作。

#### 交互流程

```
选择 Bot → 查看会话列表 → 点击查看详情 → 重置/挂起/查看上下文
```

#### 会话状态

| 状态 | 标识 | 说明 |
|------|------|------|
| 活跃 | 🟢 活跃 | 当前正在使用的会话 |
| 已重置 | 🔴 已重置 (idle/daily) | 被自动重置的旧会话 |
| 过期 | ⚪ 过期 | 超过 90 天保留期的会话 |
| 挂起 | 🟡 挂起 | 被手动挂起的会话 |

#### 操作

| 操作 | 说明 | 确认 |
|------|------|------|
| 查看上下文 | 跳转到上下文详情页 | 无 |
| 查看记忆 | 跳转到记忆详情页 | 无 |
| 重置会话 | 清空工作记忆，创建新 session_id | 二次确认 |
| 挂起会话 | 标记为挂起，下次访问强制重置 | 二次确认 |

#### 重置策略配置

| 参数 | 说明 | 默认值 |
|------|------|--------|
| 重置模式 | daily/idle/both/none | both |
| 每日重置时间 | 小时 (0-23) | 4 |
| 空闲超时 | 分钟 | 1440 (24h) |
| 重置通知 | 是否发送通知 | true |

### 4.3 上下文详情

#### 功能描述

展示当前会话的完整上下文结构，帮助理解 Bot 的"思维链"。

#### 展示层级

```
┌─────────────────────────────────────────┐
│ System Prompt                            │ ← 只读，展示人格配置
├─────────────────────────────────────────┤
│ 工作记忆 (Working Memory)               │ ← 可展开/收起
│   - 历史摘要                             │
│   - 原始消息                            │
├─────────────────────────────────────────┤
│ 情景记忆召回 (Episodic Recall)          │ ← 可展开/收起
│   - 相关记忆列表（按相关性排序）         │
├─────────────────────────────────────────┤
│ 语义记忆 (Semantic Facts)              │ ← 可展开/收起
│   - 用户画像结构化展示                  │
├─────────────────────────────────────────┤
│ System Suffix                           │ ← 动态注入的上下文摘要
└─────────────────────────────────────────┘
```

#### 上下文压缩

| 指标 | 说明 |
|------|------|
| 压缩次数 | 本会话触发压缩的次数 |
| 最后压缩 | 最近一次压缩的时间 |
| 节省比例 | 原始 → 压缩后的字符数减少比例 |
| 压缩配置 | 硬上限/软上限设置值 |

#### 数据来源

```python
# Rust 调用 Python Core
context = memory_engine.load_context(current_input)
# 返回:
# - working_history: 摘要 + 原始消息
# - episodic_recall: 向量召回结果
# - semantic_facts: 用户画像
# - system_suffix: 动态摘要
```

### 4.4 日志查看

#### 功能描述

分页查看 AI Companion 的运行日志，支持筛选、搜索、实时流。

#### 日志类型

| 类型 | 标签色 | 说明 |
|------|--------|------|
| 对话 | 绿色 | user ↔ bot 消息 |
| 会话 | 黄色 | Session 创建/重置/挂起 |
| 记忆 | 紫色 | 记忆加载/存储/抽取/压缩 |
| 主动 | 粉色 | 主动唤醒判断/发送 |
| API | 青色 | 模型调用/Token消耗 |
| 错误 | 红色 | API错误/重试/异常 |

#### 日志级别

| 级别 | 色值 | 说明 |
|------|------|------|
| DEBUG | 灰色 | 调试信息 |
| INFO | 白色 | 一般信息 |
| WARN | 黄色 | 警告信息 |
| ERROR | 红色 | 错误信息 |

#### 筛选器

| 筛选项 | 类型 | 说明 |
|--------|------|------|
| 级别 | 单选 | all/info/warn/error |
| 类型 | 单选 | all/对话/记忆/会话/主动/API/错误 |
| 日期 | 单选 | today/yesterday/指定日期 |
| 搜索 | 文本 | 关键词匹配（支持空格分隔多词） |

#### 实时日志

- 开关：Toggle Switch，控制 WebSocket 连接
- 连接状态：圆点指示（绿色=已连接，灰色=未连接）
- 数据保留：最新 500 条在内存，超出后丢弃旧数据

#### 日志导出

- 格式：JSONL（每行一个 JSON）
- 内容：包含所有字段（时间、级别、类型、消息、详情）
- 文件名：`ai-companion-{bot_id}-{date}.jsonl`

### 4.5 记忆管理

#### 功能描述

查看 Bot 的三层记忆体系，手动管理记忆条目。

#### 三层记忆

| 层级 | 存储 | 内容 | 用途 |
|------|------|------|------|
| 工作记忆 | SQLite | 原始对话消息 | 当前会话 |
| 情景记忆 | SQLite + Chroma | 重要情景片段 | 跨会话召回 |
| 语义记忆 | SQLite | 用户事实/偏好 | 用户画像 |

#### 记忆详情

**工作记忆**
- 按 Session 分组
- 显示消息列表（role、content、timestamp）
- 历史摘要折叠展示

**情景记忆**
- 按相关性/时间排序
- 显示摘要 + 重要性星级
- 支持关键词搜索

**语义记忆**
- 结构化分类展示：
  - 💕 关系相关（关系状态、好评度、里程碑）
  - 👤 基本信息（职业、年龄、城市）
  - ❤️ 喜好相关（喜欢、讨厌）
  - 🎂 重要日期（生日、纪念日）

#### 操作

| 操作 | 范围 | 确认 |
|------|------|------|
| 查看 | 单条 | 无，点击展开 |
| 删除 | 单条 | 二次确认 |
| 标记重要 | 单条 | 无 |
| 清空全部 | 全部 | 二次确认 + 输入 Bot ID |

### 4.6 设置面板

#### 功能描述

配置 AI Companion 的运行参数。

#### 配置分组

**模型配置**
| 字段 | 类型 | 说明 |
|------|------|------|
| Provider | 下拉 | minimax/openai/claude/ollama/custom |
| API Key | 密码输入 | 加密存储 |
| Base URL | 文本输入 | API 端点 |
| Model | 文本输入 | 模型名称 |
| Temperature | 数字输入 | 0.0-2.0 |
| Max Tokens | 数字输入 | 最大输出 |

**平台配置**
| 平台 | 状态 | 配置项 |
|------|------|--------|
| CLI | 始终启用 | - |
| 飞书 | 启用/禁用 | App ID, App Secret, 连接模式 |
| Webhook | 启用/禁用 | Host, Port, Path |

**上下文配置**
| 字段 | 类型 | 说明 |
|------|------|------|
| 最大工作记忆轮数 | 数字 | 默认 20 |
| 上下文硬上限 | 数字 | 字符数，默认 5000 |
| 上下文软上限 | 数字 | 字符数，默认 3000 |
| 结构化压缩 | 开关 | 启用/禁用 |
| 触发阈值 | 数字 | 0.0-1.0 |
| 尾部保留 | 数字 | tokens |

**主动唤醒配置**
| 字段 | 类型 | 说明 |
|------|------|------|
| 启用 | 开关 | 是否启用主动唤醒 |
| 空闲阈值 | 数字 | 小时 |
| 最小间隔 | 数字 | 小时 |
| 每日最大 | 数字 | 条 |
| 情绪关键词 | 标签输入 | 逗号分隔 |

#### 保存与重置

- **保存**：验证输入 → 调用 Tauri Command → 写入配置文件 → 显示成功 toast
- **重置**：恢复到默认值 → 二次确认

#### 危险操作

| 操作 | 影响 | 防护 |
|------|------|------|
| 清空记忆 | 删除所有记忆数据 | 二次确认 + 输入 Bot ID |
| 重置配置 | 恢复默认配置 | 二次确认 |

## 七、数据流

### 7.1 监控数据流

```
┌─────────────┐     5秒轮询      ┌─────────────┐
│   Rust      │ ──────────────→ │   React     │
│  Backend    │   JSON 指标     │   Frontend  │
│ (sysinfo)   │ ←────────────── │  (Zustand)  │
└─────────────┘    调用命令      └─────────────┘
       ↑
       │ Python Core
       ↓
┌─────────────┐
│  SQLite /  │
│  Files      │
└─────────────┘
```

### 7.2 日志数据流

```
┌─────────────┐   WebSocket    ┌─────────────┐
│   Rust      │ ──────────────→ │   React     │
│  Backend    │   实时日志流    │   Frontend  │
│ (tail -f)  │ ←─────────────── │  (Zustand)  │
└─────────────┘    HTTP 轮询   └─────────────┘
       ↑                              ↑
       │                              │
       └──────────┐   分页请求       │
                  ↓                   │
            ┌─────────────┐           │
            │  日志文件    │ ←─────────┘
            │  (JSONL)    │
            └─────────────┘
```

### 7.3 配置数据流

```
用户编辑 → React Form → Tauri Command → Python ConfigLoader → YAML 文件
                                ↓
                         验证输入
                                ↓
                         保存成功 toast
```

## 八、边界情况

### 8.1 监控页面

| 情况 | 处理 |
|------|------|
| Bot 未运行 | 显示"未运行"状态，数据置灰 |
| API 连接失败 | Token 消耗显示"--"，红色提示 |
| Token 超限 | 卡片变红，显示警告 |

### 8.2 会话管理

| 情况 | 处理 |
|------|------|
| 无会话 | 显示空状态："暂无会话记录" |
| 会话过期 | 显示过期标签，不可操作 |
| 重置失败 | 显示错误 toast |

### 8.3 日志查看

| 情况 | 处理 |
|------|------|
| 日志文件不存在 | 显示空状态 |
| 搜索无结果 | 显示"未找到匹配的日志" |
| WebSocket 断开 | 自动重连，最多重试 3 次 |

### 8.4 配置保存

| 情况 | 处理 |
|------|------|
| API Key 格式错误 | 输入框下方红色提示 |
| 测试连接失败 | 显示错误详情，允许跳过 |
| 保存冲突 | 显示"配置已被修改"，提供覆盖选项 |

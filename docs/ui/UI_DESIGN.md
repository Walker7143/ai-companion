# AI Companion 管理后台 - UI 设计方案（网页版）

> 版本: v2.0
> 日期: 2026-04-26
> 变更: 从 Tauri 桌面端改为网页端

---

## 一、产品概述

### 1.1 产品定位

AI Companion 管理后台 — 面向用户的网页控制面板，用于监控 Bot 状态、管理会话和上下文、查看日志、配置参数。

### 1.2 核心用户

- 普通用户：管理自己的 AI 伴侣
- 开发者：调试 Bot 行为、查看日志

### 1.3 使用场景

- 网页端为主（浏览器访问）
- 桌面和移动端均可使用（响应式）

---

## 二、设计原则

### 2.1 视觉定位

**类比产品：** Linear / Notion / Vercel Dashboard

- 简洁、现代、高效
- 亮色主题为主（清晰易读）
- 暗色主题可选

### 2.2 设计原则

| 原则 | 说明 |
|------|------|
| **简洁清晰** | 减少视觉噪音，突出内容 |
| **一致体验** | 全局统一的交互模式 |
| **快速响应** | 操作即时反馈 |
| **信息分层** | 重要信息突出，次要信息收敛 |

### 3.2 字体系统

| 用途 | 字体 | 大小 |

| 下拉菜单 | `0 4px 16px rgba(0,0,0,0.12)` |
| 弹窗 | `0 8px 32px rgba(0,0,0,0.16)` |

## 四、页面布局

### 4.1 整体布局

```
┌─────────────────────────────────────────────────────────────┐
│  Header（固定顶部）                                          │
│  ┌──────┬────────────────────────────────────────────────┐ │
│  │      │                                                │ │
│  │ 侧边栏 │           主内容区                              │ │
│  │       │                                                │ │
│  │ 导航  │                                                │ │
│  │       │                                                │ │
│  └──────┴────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────┘
```

- **Header**：高度 56px，固定定位，包含 Logo、Bot 切换器、主题切换、用户菜单
- **侧边栏**：宽度 240px，可折叠，包含导航菜单
- **主内容区**：自适应宽度，内边距 24px，最大宽度 1200px

### 4.2 响应式断点

| 断点 | 宽度 | 布局变化 |
|------|------|----------|
| 桌面 | ≥1024px | 侧边栏展开 |
| 平板 | 768px-1023px | 侧边栏收起为图标 |
| 移动 | <768px | 侧边栏隐藏，汉堡菜单 |

## 五、页面设计

### 5.1 监控面板（Dashboard）

**布局：**
- 顶部标题区 + 刷新按钮
- 4 列统计卡片网格
- 2 列系统指标卡片
- 会话统计详情卡片

**统计卡片样式：**
- 白色背景 + 轻阴影
- 左侧图标（圆形浅色背景）
- 数字用大字号 + 粗体
- 副标题用弱色

### 5.2 会话管理（Session）

**布局：**
- 顶部筛选器（平台/状态/日期）
- 会话列表（卡片式）
- 右侧可选会话详情面板

**会话卡片：**
- 平台图标 + 会话标题
- 状态标签（运行中/已暂停/已结束）
- 时间 + Token 统计
- 操作按钮（重置/挂起/查看详情）

### 5.3 日志查看（Logs）

**布局：**
- 顶部筛选栏（级别/类型/日期/搜索）
- 日志列表（表格或卡片式）
- 底部分页器

**日志条目样式：**
- 时间戳（弱色）
- 级别标签（彩色圆点 + 文字）
- 类型标签
- 消息内容（可展开详情）

### 5.4 记忆管理（Memory）

**布局：**
- Tab 切换（统计概览 / 工作记忆 / 情景记忆 / 语义记忆）
- 统计概览：三层记忆环形图 + 数字统计
- 各记忆类型：列表展示，可搜索/删除

### 5.5 设置（Settings）

**布局：**
- 分区块卡片（模型配置 / 平台配置 / 上下文配置 / 主动唤醒）
- 表单式输入
- 底部保存/重置按钮

## 六、组件规范

### 6.1 按钮

| 类型 | 样式 |
|------|------|
| 主按钮 | 紫色背景，白色文字，悬停加深 |
| 次按钮 | 白色背景，紫色边框，悬停浅紫背景 |
| 危险按钮 | 红色背景，悬停加深 |
| 幽灵按钮 | 透明背景，悬停浅灰背景 |

**尺寸：** 小（28px）/ 中（36px）/ 大（44px）

### 6.2 输入框

- 白色背景，灰色边框
- 聚焦时紫色边框 + 阴影
- 错误时红色边框
- 圆角 6px，高度 36px（中）/ 44px（大）

### 6.3 卡片

- 白色背景（亮色主题）
- 轻阴影 + 1px 边框
- 圆角 8px
- 内边距 16px 或 20px

### 6.4 标签/徽章

- 圆角 4px
- 浅色背景 + 深色文字
- 或深色背景 + 浅色文字（突出时）

### 6.5 表格

- 表头：背景 #F8FAFC，文字加粗
- 行：白色背景，悬停浅灰
- 分割线：浅灰色
- 圆角 8px

### 6.6 空状态

- 居中图标（64px，弱色）
- 标题 + 描述
- 可选操作按钮

### 6.7 骨架屏

- 浅灰色块
- 圆角与实际组件一致
- 微妙的脉动动画

## 七、动效规范

| 场景 | 动效 |
|------|------|
| 页面切换 | 淡入 150ms |
| 卡片悬停 | 阴影加深 150ms |
| 按钮点击 | 缩放 0.98 50ms |
| 下拉展开 | 最大高度动画 200ms |
| 侧边栏收起 | 宽度动画 200ms ease-out |
| Toast 出现 | 从右侧滑入 200ms |
| Modal 打开 | 缩放 0.95→1 + 淡入 200ms |

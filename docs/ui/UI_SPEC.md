# AI Companion 管理后台 - 产品规格（网页版）

> 版本: v2.0
> 日期: 2026-04-26

---

## 一、产品概述

### 1.1 定位

AI Companion 管理后台是一个网页应用，为用户提供便捷的 AI 伴侣管理界面。

### 1.2 核心功能

1. **监控面板** - 实时查看 Bot 运行状态、系统资源、记忆统计
2. **会话管理** - 查看和管理与 Bot 的对话记录
3. **日志查看** - 浏览、筛选、搜索对话和系统日志
4. **记忆管理** - 查看和管理三层记忆系统（工作/情景/语义）
5. **设置配置** - 配置模型参数、平台设置、主动唤醒规则

### 1.3 非功能需求

- 响应式设计，支持桌面/平板/移动端
- 亮色/暗色主题切换
- 数据实时刷新

## 二、页面详细规格

### 2.1 监控面板（Dashboard）

#### 功能清单

| 功能 | 描述 |
|------|------|
| 统计卡片 | 4 个：今日会话、运行时长、记忆总数、Bot状态 |
| 系统指标 | CPU 使用率 + 内存使用率（含进度条） |
| 会话统计 | 今日会话数、主动消息数、Token消耗 |
| 记忆分布 | 工作/情景/语义记忆数量 |

#### 交互

- 5 秒自动刷新数据
- 手动刷新按钮
- Bot 切换器切换不同 Bot

#### 数据来源

- `get_system_metrics` - CPU/内存
- `get_bot_metrics` - Bot 运行时统计

---

### 2.2 会话管理（Session）

#### 功能清单

| 功能 | 描述 |
|------|------|
| 会话列表 | 平台图标、标题、状态、时间、Token |
| 平台筛选 | 全部/CLI/飞书 |
| 状态筛选 | 全部/运行中/已暂停 |
| 会话详情 | 基本信息、Token消耗、预估费用 |
| 操作 | 重置会话、挂起会话、查看详情 |

#### 交互

- 点击会话卡片展开详情
- 重置/挂起需二次确认
- 详情面板右侧滑出

#### 数据来源

- `list_sessions`
- `get_session_detail`
- `reset_session`
- `suspend_session`

---

### 2.3 日志查看（Logs）

#### 功能清单

| 功能 | 描述 |
|------|------|
| 级别筛选 | 全部/info/warn/error |
| 类型筛选 | 全部/dialogue/session/api/proactive |
| 日期筛选 | 日期选择器 |
| 关键词搜索 | 模糊匹配消息内容 |
| 日志列表 | 时间、级别、类型、平台、消息 |
| 分页 | 每页 20 条 |
| 实时日志 | Toggle 开关 + WebSocket 流 |
| 日志导出 | JSON/CSV 格式 |

#### 交互

- 开启实时日志后新日志自动追加
- 点击日志条目展开详情
- 导出按钮下载文件

#### 数据来源

- `get_logs`
- `stream_logs`
- `export_logs`

---

### 2.4 记忆管理（Memory）

#### 功能清单

| 功能 | 描述 |
|------|------|
| Tab 切换 | 统计概览/工作记忆/情景记忆/语义记忆 |
| 统计概览 | 三层记忆环形图 + 数字统计 |
| 工作记忆 | 消息列表，按会话分组 |
| 情景记忆 | 重要性星级 + 摘要 + 详细内容 |
| 语义记忆 | 用户画像、关系状态、态度评分、关键事实 |
| 搜索 | 关键词搜索记忆 |
| 删除 | 删除单条记忆（需确认） |
| 清空 | 清空全部记忆（需二次确认） |

#### 数据来源

- `get_memory_stats`
- `get_working_memory`
- `get_episodic_memory`
- `get_semantic_memory`
- `delete_memory`
- `clear_all_memory`

---

### 2.5 设置（Settings）

#### 功能清单

| 功能 | 描述 |
|------|------|
| 模型配置 | Provider、API Key、Base URL、Model、Temperature、Max Tokens |
| 平台配置 | CLI/飞书/Webhook 开关 |
| 上下文配置 | Hard Limit、Soft Limit、Max Working Turns |
| 主动唤醒配置 | 启用开关、空闲阈值、最小间隔、每日最大次数、情绪关键词 |
| 保存配置 | 持久化到后端 |
| 重置配置 | 恢复默认值 |
| 测试连接 | 测试 API 连通性 |

#### 数据来源

- `get_config`
- `update_config`
- `test_api_connection`

## 三、通用交互规范

### 3.1 导航

- 左侧导航栏 5 个菜单项
- 当前页面高亮显示
- 点击跳转页面

### 3.2 Bot 切换

- Header 下拉选择器
- 切换后刷新页面数据

### 3.3 主题切换

- Header 图标按钮
- 切换立即生效
- 偏好保存到 localStorage

### 3.4 加载状态

- 骨架屏占位
- 按钮 Loading spinner
- Toast 提示加载中

### 3.5 错误处理

- Toast 显示错误信息
- 可重试操作显示重试按钮
- 网络错误自动提示

### 3.6 空状态

- 无数据时显示空状态插图
- 友好说明文字
- 可选操作引导

## 四、验收标准

### 4.1 功能验收

- [ ] 所有页面可正常访问
- [ ] 数据能正确加载和显示
- [ ] 筛选/搜索功能正常
- [ ] 操作有确认提示
- [ ] 主题切换正常
- [ ] Bot 切换后数据刷新

### 4.2 交互验收

- [ ] 页面加载有骨架屏过渡
- [ ] 按钮点击有反馈
- [ ] 表单提交有 Loading 状态
- [ ] 操作成功/失败有 Toast 提示

### 4.3 响应式验收

- [ ] 桌面端（≥1024px）：侧边栏展开
- [ ] 平板端（768-1023px）：侧边栏收起为图标
- [ ] 移动端（<768px）：侧边栏隐藏，汉堡菜单

### 4.4 性能验收

- [ ] 首屏加载 < 2 秒
- [ ] 页面切换流畅
- [ ] 数据刷新无闪烁
